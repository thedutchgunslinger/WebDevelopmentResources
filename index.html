Custom Cursor

Voorbeeld van Custom Cursor

Bekijk op Codepen

Uitleg

Om te beginnen moeten we een lege div aan maken in onze HTML bestand.

Nu moeten we de cursor nog mooi maken. In dit geval heb ik gekozen voor een simpel rondje. We gebruiken een CSS id om de cursor mooi te maken.

Nu hebben we een rondje op ons scherm die nog niet beweegt. In ons javascript bestand kunnen we de cursor onze muis laten volgen. Om dit te berijken moeten we eerst de div van de cursor ophalen. Daarna moeten we een functie maken die de muis positie ophaalt en dan de cursor positie update.

Nu volgt de cursor de muis. Als je een soort smooth volg effect wilt dan moet in de CSS van de cursor een transition delay erin zetten.

Muis Parallax Effect

Voorbeeld van het muis parallax effect

Bekijk op Codepen

Uitleg

In dit geval beginnen we bij de javascript, we gaan werken met data attributes in html, maar dit moeten we eerst aangeven in onze javascript. om te beginnen gaan we netzoals bij de custom cursor een event listener laten luisteren naar de muis update zodat we de muis locatie kunnen ophalen. En iedere keer als de muis positie veranderd willen de functie parallax oproepen.

Nu gaan we de functie parallax schrijven. In de functie gaan we eerst alle elementen ophalen met de class, in dit geval .vierkant, en gaan we een stukje code bij ieder element runnen.

Het zou cool zijn als we in de html tag kunnen aangeven hoe heftig de beweging moet zijn zodat niet alles het zelfde beweegt. Dit doen we met een data attribute, dit kunnen we in de html tag zetten en dan in de javascript ophalen.

Om er voor te zorgen dat de beweging klein blijft en aangepast word door de data attribute moeten we een kleine bereking doen. De bereking moeten we doen voor zowel de x as als de y as. Door het te delen door 250 zorgen we ervoor de beweging klein blijft. Daarna passen we de x en y positie aan van het element.

In de html kunnen we nu de data attribute gebruiken in combinatie met de class.

CSS Variabelen

Voorbeeld van CSS Variabelen

Bekijk op Codepen

Uitleg

Om het aanpassen van onze css wat makkelijk te maken kunnen we gebruik maken van css variabelen, hierdoor hoeven we maar een waarde aan te passen en voor bijvoorbeeld een kleur aan te passen i.p.v. bij iedere regel css waar die kleur voorkomt.

Om te beginnen moeten we eerst een variabele aanmaken. Dit doe je in de :root van je css.

Nu we een vabele aan hebben gemaakt kunnen we hem gebruiken in de rest van onze css. Dit doen we doormiddel van de functie var().
